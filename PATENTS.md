# Patent Notes

HTMLCut's own code is licensed under the MIT License, which does not include
an explicit patent grant or patent retaliation clause.

## Dependency Patent Grants

The compiled HTMLCut binary includes crates under several licenses with
different patent postures:

| License | Explicit Patent Grant | Notable Crates |
|:--------|:----------------------|:---------------|
| MIT | No (implicit only) | HTMLCut, scraper, ego-tree, and many others |
| Apache-2.0 | Yes | html5ever, serde, regex, ureq, url, clap, and others |
| MPL-2.0 | Yes (within MPL scope) | cssparser, selectors, servo_arc |
| ISC | No | ring, rustls-webpki |
| Unicode-3.0 | No | ICU data crates |
| CDLA-Permissive-2.0 | No | webpki-root-certs |

Apache-2.0 includes an explicit patent grant in Section 3 from each contributor
to those respective crates. MPL-2.0 includes a patent grant in Section 2.1
scoped to the licensed files.

## Patent Non-Assertion

The copyright holder (Ervins Strauhmanis) makes no patent claims on HTMLCut's
implementation and is not aware of any patents it infringes.

This is a statement of intent, not a formal legal instrument.
